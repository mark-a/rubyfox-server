# Rubyfox::SFS

SmartFox Server bundled as a gem.

## Installation

    $ gem install rubyfox-sfs

## Usage

    $ smartfox install /path/to/smartfox

    $ smartfox config /path/to/smartfox /path/to/template/dir

    $ smartfox start /path/to/smartfox
